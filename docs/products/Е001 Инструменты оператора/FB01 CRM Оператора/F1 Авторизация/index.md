# Описание F1 (Авторизация)

## Назначение

Capability "Payments" отвечает за все операции, связанные с приемом и обработкой платежей в Product A. Включает в себя функции оплаты, возвратов, управления платежными методами и интеграции с платежными провайдерами.

## Границы ответственности

**Входит в scope:**

- Прием платежей от клиентов
- Обработка возвратов (полных и частичных)
- Управление платежными методами пользователя
- Интеграция с платежными шлюзами
- Формирование чеков и отчетности по транзакциям

**Не входит в scope:**

- Управление подписками (отдельный capability Subscriptions)
- Бухгалтерский учет (отдельный capability Accounting)
- Fraud-мониторинг (отдельный capability Risk Management)

## Связанные функции (Features)

| Feature | Статус | Описание |
|---------|--------|----------|
| [Refund](features/refund/index.md) | Active | Возврат средств клиентам |
| Checkout | Active | Оформление оплаты |
| Payment Methods | Planned | Управление способами оплаты |

## Интеграции

- **Payment Gateway** — основной платежный шлюз для обработки транзакций
- **Notification Service** — отправка уведомлений о статусе платежа
- **Order Service** — получение информации о заказах для оплаты
